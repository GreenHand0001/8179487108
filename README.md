# 项目简介

## 核心功能
- 基于YOLOv8的纸箱包裹检测系统
- 支持图片、视频及摄像头进行检测
- 支持选择不同检测模型
- 实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并在界面直接查看所有检测结果
- 支持检测结果保存
- 支持自定义置信度和IOU阈值
- 可选择显示标签和原图

## 适用场景
- 包裹检测与分类
- 物流分拣系统
- 仓库管理
- 生产质量控制
- 自动化生产线监控

## 技术特点
- 使用Python3.9开发
- 环境依赖：opencv-python、PyQt5、ultralytics、torch1.9
- 内置3000张数据集（1类：package）
- 包含完整UI界面（Pycharm+Anaconda或VSCode+Anaconda）
- 提供完整源码、标注数据集、训练好的模型及结果
- 配置成功环境可直接使用
